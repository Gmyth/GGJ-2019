# Pillow WarGGJ-2019

This is a game jam project which came from GGJ2019 and get it done by 48 hours.

Just play with your friend and throw pillows to them! And you win!

Support at most 4 player play together. 4 different type of xbox controller input allowed, and 2 types keyboard input.
